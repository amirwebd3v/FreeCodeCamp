# JavaScript Algorithms and Data Structures Projects

A collection of interactive JavaScript projects demonstrating various algorithms and practical applications. Each project showcases different aspects of JavaScript programming and data manipulation.

## 🌐 Live Demo

The project is deployed and can be viewed at: [https://fcc-js-algo-amirwebdev.vercel.app](https://fcc-js-algo-amirwebdev.vercel.app)

## 📂 Projects

| Project | Description | Link |
|---------|-------------|------|
| Palindrome Checker | An interactive tool to verify if words or phrases read the same backward as forward. Includes examples like "race car" and "never odd or even". | [Try Palindrome Checker](https://fcc-js-algo-amirwebdev.vercel.app/palindrome-checker.html) |
| Phone Number Validator | A utility to validate US phone numbers in various formats, supporting multiple valid patterns including area codes and different separators. | [Try Phone Validator](https://fcc-js-algo-amirwebdev.vercel.app/phone-number-validator.html) |
| Roman Numeral Converter | A conversion tool that transforms numbers (1-3999) into Roman numerals, complete with a comprehensive guide for Roman numeral symbols. | [Try Roman Converter](https://fcc-js-algo-amirwebdev.vercel.app/roman-numeral-converter.html) |
| Cash Register | An interactive cash register simulator that calculates change and manages cash drawer operations. | [Try Cash Register](https://fcc-js-algo-amirwebdev.vercel.app/cash-register.html) |
| Pokémon Search | A Pokédex-style application for searching and exploring Pokémon stats, types, and detailed information. | [Try Pokémon Search](https://fcc-js-algo-amirwebdev.vercel.app/pokemon-search.html) |

## 🛠️ Technologies Used

## 🚀 Getting Started

To run this project locally:

1. Clone the repository:
```bash
git clone https://github.com/amirwebd3v/freeCodeCamp.git
```

2. Navigate to the project directory:
```bash
cd javascript-algorithms
```

3. Since this is a static site, no build step is required. Simply open any HTML file in your browser to view the project.


## 👨‍💻 Development

The project structure is straightforward with separate HTML files for each tool. The deployment configuration is handled through Vercel with static file serving.
