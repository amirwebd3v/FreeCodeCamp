Introduction

Python is a high-level, interpreted programming language known for its simplicity and readability.

Created by Guido van Rossum and released in 1991, Python emphasizes code readability with its notable use of significant whitespace.

Python features a dynamic type system and automatic memory management.

print("Hello, World!")
Getting Started

To start programming in Python, you'll need to:

  • Download Python from python.org
  • Install it on your system
  • Choose a code editor or IDE

You can verify your Python installation by opening a terminal and typing:

python --version

Python comes with a simple IDE called IDLE, but many developers prefer alternatives like VS Code or PyCharm.

Data Types

Python has several built-in data types:

  • Numbers (int, float, complex)
  • Strings (str)
  • Lists
  • Tuples
  • Dictionaries
x = 42 # integer y = "Hello" # string z = [1, 2, 3] # list

Python's dynamic typing means you don't need to declare variable types explicitly.

Control Flow

Control flow statements determine the order in which program code executes.

Python provides several control flow tools:

if condition: # code block elif another_condition: # code block else: # code block

Loops are another important control flow tool:

for item in items: # code block while condition: # code block
Functions

Functions are blocks of reusable code that perform specific tasks.

To define a function in Python, use the def keyword:

def greet(name): return f"Hello, {name}!"

Functions can take parameters and return values.

Python also supports lambda functions for simple operations:

square = lambda x: x * x
  • Functions should be small and focused
  • Use clear and descriptive names
  • Include docstrings for documentation
